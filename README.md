# demo-tools

CLI + Copier template for deploying demo web apps to Fly.io.

See `2026-04-29-demo-deployment-template/design.md` in the `tasks` repo for the full design.

## Install

```bash
uv tool install git+https://github.com/BosTheCoder/demo-tools
```

## Usage

```bash
demo-init nextjs my-demo
cd my-demo && just deploy
```
